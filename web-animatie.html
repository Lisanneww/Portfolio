Minor Visual Interface Design

Web-animatie

Bij dit vak zijn we een stap verder gegaan om een statische illustratie te animeren in code. Zo heb ik gekozen voor acht geometrische vogels. Deze heb ik geïllustreerd en geëxporteerd als een SVG bestand. Ik heb de volgens verschillende functies gegeven met code zoals: zweven, vliegen, kleur verandering. Bovendien was de site responsive en heb ik extra wolk animaties toegevoegd. Ik heb heel veel geleerd tijdens dit vak. Ik vond het moeilijk om te beginnen aangezien ik totaal geen ervaring had met coderen, maar dat maakte het juist wel spannend.
